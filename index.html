Machine Learning Research Assistant

I am currently working with a team to make an ai for the purpose of real time epidemiological monitoring. My involvement has been primarily in making a 'symptom predictor' which takes a natural language input and outputs a list of symptoms that were endorsed in the input. This model is getting well over 99% accuracy with a positive predictive value of around 88% (note that we are predicting a sparse matrix), and was implemented using data of over a million samples. Master's Thesis

The thesis project for my Master's was centered on surveying many different machine learning implementations for predicting height from genetic data. The main purpose of this project was to see if a neural network approach could outperform a typical statistical learning approach that is used in practice (lasso regression). None of the other models that we attempted could out perform lasso regression, but one novel approaches I applied as well as the lasso regression implementation can be found by clicking the 'learn more' button.
